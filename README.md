# E-Commerce Sales Analysis

## Data Source
Chen, D. (2012). Online Retail II [Dataset]. UCI Machine Learning Repository. https://doi.org/10.24432/C5CG6D.

**Link**: [Online Retail II](https://archive.ics.uci.edu/dataset/502/online+retail+ii)
